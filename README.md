# 2D 3D Hybrid Action Platformer

A 3D action platformer built in **Unreal Engine 5.4**, using 2D sprites in a stylized pirate world. You play as a fearless pig pirate battling through hordes of shark pirates.

---

## 🎮 Gameplay

- Take control of the **Pig Pirate**, a brave swashbuckler on a quest for glory.
- **Defeat shark pirates** using melee attacks and special abilities.
- Collect **power-ups** as you progress:
  - 🗡️ Throwing Sword – attack from a distance
  - 🐷 Double Jump – reach higher platforms
- Navigate through each level by dodging enemies and reaching the **end goal** to complete the stage.

---

## 🛠 Engine & Features

- Built with **Unreal Engine 5.4**
- Hybrid 3D world using **2D flipbook sprites**
- Custom character controller with platforming mechanics
- Power-ups

---

## 🚀 How to Play

1. Clone or download the project.
2. Open the `.uproject` file in **Unreal Engine 5.4**.
3. Press `Play` to start testing the game inside the editor.

---

## 📁 Project Structure

- `Content/` – Game assets, sprites, animations, blueprints
- `Config/` – Project and engine settings
- `.uproject` – Project file to open in Unreal Engine

> Auto-generated folders like `/Intermediate`, `/Saved`, and `/Binaries` are excluded.
